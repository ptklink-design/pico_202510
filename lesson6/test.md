# Streamlit & MQTT 監控程式

## 專案簡介

這是一個基於 Streamlit 和 MQTT 協議的即時監控系統。本程式作為 MQTT 訂閱者，接收來自 IoT 設備的感測器數據，並透過 Streamlit 提供即時監控與數據視覺化介面。

## 功能需求

### 1. 電燈開關狀態顯示
- 即時顯示電燈的開關狀態
- 以視覺化方式呈現（如開關按鈕、狀態指示燈等）

### 2. 客廳溫度監控
- 即時顯示客廳的溫度數值
- 支援溫度單位切換（攝氏/華氏）

### 3. 客廳濕度監控
- 即時顯示客廳的濕度數值
- 顯示濕度百分比

### 4. 溫濕度圖表視覺化
- 使用圖表（折線圖/區域圖）顯示溫濕度歷史數據
- 支援即時更新圖表
- 可選擇顯示時間範圍

### 5. 時間軸設定
- X 軸顯示日期與時間
- 支援時間格式自訂
- 可縮放時間範圍

### 6. 數據匯出功能
- 支援將溫濕度數據匯出為 CSV 檔案
- 可選擇匯出的時間範圍
- 包含完整的時間戳記和數據欄位

## 技術架構

- **前端框架**: Streamlit
- **通訊協議**: MQTT (paho-mqtt)
- **數據視覺化**: Streamlit 內建圖表組件
- **數據格式**: JSON / CSV

## MQTT 主題設定

- 電燈開關: `home/light/status`
- 客廳溫度: `home/livingroom/temperature`
- 客廳濕度: `home/livingroom/humidity`

## 使用說明

1. 啟動 MQTT Broker
2. 執行 Streamlit 應用程式
3. 開啟瀏覽器查看監控介面
4. 訂閱 MQTT 主題接收數據
5. 即時查看感測器數據和圖表
6. 需要時匯出 CSV 檔案
